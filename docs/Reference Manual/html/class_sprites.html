Detailed Description

This class is a data layer class composed within the Presentation class. it stores all the sprite objects that the Presentation class draws on the screen. This class also encapsulates the SpriteFactory class.

This method receives three constant enumerations. The first two are of type MapKeys, they represent the key values of the objects in the sprite map that are of the same object type but have different orientations. The third enumeration is of type orientation and it describes the orientation of the object that this method will move. In addition, the method also receives a tuple which describes the coordinates that the object will move to.
